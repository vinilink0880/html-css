Atalhos de VS Code

Neste artigo espero ajudar as pessoas que usam o VS Code a codificar ainda mais rapidamente.

Nem todo mundo tem tempo para seguir todas as dicas e truques para encontrar os atalhos que os ajudam a codificar mais rápido — há tantos deles.

Então, vou listar meus atalhos favoritos que me tornam um codificador veloz.

Vamos começar.

Unir Linhas

Unir Linhas

  • No Mac: Ctrl+J
  • No Ubuntu, Windows: Abra os atalhos de teclado em Arquivo> Preferências > Atalhos do teclado, e vincule action.joinLines para um atalho de sua escolha.

Formatar Código

Este atalho ajuda a indentar o código como já definido nas configurações do editor.

Formatar código

Dica: Use-o no fim ou no início. Prefiro usá-lo quando termino o código em um arquivo.

  • No Windows: Shift + Alt + F
  • No Mac: Shift + Option + F
  • No Ubuntu: Ctrl + Shift + I

Encontrar e eliminar espaço em branco

Este comando ajuda a eliminar espaços em branco extras no início e no final e te poupa de qualquer tipo de Lint mais rigoroso.

Eliminar espaço em branco

Dica: Use-o no final, quando terminar o código em um arquivo.

Como alternativa, você pode ativar a eliminação de espaços em branco a partir das próprias configurações:

  1. Abra as Configurações de Usuário no VS (Preferências > Configurações > Aba de Configurações do Usuário).
  2. Clique no ícone {} na parte superior direita da janela. Isso abrirá um documento.
  3. Adicione uma nova configuração “files.trimTrailingWhitespace”: true para os documentos de Configurações do Usuário, se ele ainda não estiver lá. Isso significa que você não está editando a configuração padrão diretamente — mas sim adicionando uma nova.
  4. Salve o arquivo de Configurações do Usuário.

Também adicionamos um novo comando para acionar isso manualmente (Trim Trailing Whitespace na paleta de comandos).

Ou, se você usar uma nova versão, siga a configuração 1 e veja a imagem abaixo.

Dobrar Código

Às vezes, se o tamanho do arquivo é grande e você apenas deseja obter uma compreensão geral do código, é necessário “dobrar” o código.

Dobrando o código

  1. Para dobrar a região não recolhida mais interna no cursor:
  • No Windows/buntu: Ctrl + Shift + [
  • No Mac: Command + Option + [
  1. O comando Unfold (“Desdobrar”) desdobra a região recolhida no cursor:
  • No Windows /Ubuntu: Ctrl + Shift + ]
  • No Mac: Command + Option + ]

Copiar Linha Para Cima/Para Baixo

Copiar linha para cima/para baixo

  • No Windows: Shift + Alt + Up/Down
  • No Mac: Shift + Option + Up/Down
  • No Ubuntu: Ctrl + Shift + Alt + Up/Down

Além disso, você pode ver as combinações de teclas associadas escolhendo: Arquivo > Preferências > Atalhos de Teclado e editando a vinculação conforme sua escolha.

Dividir Editor

  • No Windows: Shift + Alt + \ ou 2,3,4
  • No Mac: Command + \ ou 2,3,4
  • No Ubuntu: Shift + Alt + \ ou 2,3,4

Para dividir o editor, você pode usar o comando split editor (“divisão do editor”). O atalho de teclado original para um editor de divisão é 123. Ele é útil para edição lado a lado.

Além disso, você pode ver as combinações de teclas associadas acessando: Arquivo > Preferências > Atalhos do teclado e editando a vinculação conforme sua escolha.

Layout Do Editor De Grade

Por padrão, os grupos de editores são dispostos em colunas verticais (por exemplo, quando você divide um editor para abri-lo ao lado). Você pode facilmente organizar grupos de editores em qualquer layout que desejar, tanto na vertical quanto na horizontal:

Para oferecer suporte a layouts flexíveis, você pode criar grupos de editores vazios. Por padrão, fechar o último editor de um grupo de editores também fecha o próprio grupo, mas você pode alterar esse comportamento com uma nova configuração workbench.editor.closeEmptyGroups: false.

Selecionar Palavra

Selecionar palavra

  • No Windows: Ctrl + d
  • No Mac: Command + d
  • No Ubuntu: Ctrl + d

Se você pressionar o command + d mais de uma vez, você adicionará outra ocorrência da mesma palavra-chave à sua seleção.

Abrindo e Fechando a Barra Lateral

Abrindo e fechando da barra lateral

  • No Windows: Ctrl + b
  • No Mac: Command + b
  • No Ubuntu: Ctrl + b

Às vezes, quando o arquivo tem uma largura grande ou no caso de o editor de divisão fechar, a barra lateral ajuda muito.

Navegar para uma Linha Específica

Navegar para uma Linha Específica

Nota: Para ir para uma linha no arquivo, use ctrl + g, então digite um número de linha. Como alternativa, você também pode abrir o menu de arquivos com o command + p primeiro. Então digite :. Em seguida, digite o número da sua linha.

  • No Windows: Ctrl + g
  • No Mac: Command + g ou Command + p
  • No Ubuntu: Ctrl + g

Ir para Símbolo no Arquivo

  • No Windows: Ctrl + Shift + o
  • No Mac: Command + Shift + o
  • No Ubuntu: Ctrl + Shift + o

Você pode agrupar os símbolos por tipo ao adicionar dois pontos: @:.

Agrupando os símbolos

Ir para Símbolo na Área de Trabalho

Ir para Símbolo na Área de Trabalho

  • No Windows: Ctrl + t
  • No Mac: Command + t
  • No Ubuntu: Ctrl + t

Excluir Palavra Anterior

  • No Windows: Ctrl + backspace
  • No Mac: Command + delete
  • No Ubuntu: Ctrl + backspace

Isso é muito útil em situações em que você comete erros de digitação e odeia ter que pressionar e segurar o botão backspace para chegar à parte que deseja excluir.

Selecionar Palavras

  • No Windows: Ctrl + Shift + Seta para a direita / Seta para a esquerda
  • No Mac: Command + Shift + Seta para a direita / Seta para a esquerda
  • No Ubuntu: Ctrl + Shift + Seta para a direita / Seta para a esquerda

Isso é muito útil para selecionar palavras mais rapidamente e editar conforme necessário.

Duplicar Linha

  • No Windows: Ctrl + Shift + d
  • No Mac: Command + Shift + d
  • No Ubuntu: Ctrl + Shift + d

Um recurso muito poderoso e conhecido é a capacidade de duplicar linhas.

Excluindo uma linha

  • No Windows: Ctrl + x
  • No Mac: Command + x
  • No Ubuntu: Ctrl + x

Adicionar Cursor Acima/Abaixo

  • No Windows: Ctrl + Alt + Seta para cima / Seta para baixo
  • No Mac: Command + Alt + Seta para cima / Seta para baixo
  • No Ubuntu: Ctrl + Alt + Seta para cima / Seta para baixo

Duplicar seus cursores é sem dúvida o um recurso do VS Code que economiza muito tempo para você. Isso se torna ótimo em situações como o TypeScript

Renomear Símbolo

  • No Windows: F2
  • No Mac: F2
  • No Ubuntu: F2

Selecione um símbolo e digite F2. Como alternativa, você pode usar o menu de contexto.

Seleção de Coluna (Caixa)

  • No Windows: Shift + Alt
  • No Mac: Shift + Option
  • No Ubuntu: Shift + Alt

Você pode selecionar blocos de texto usando o comando acima enquanto arrasta o mouse. Um cursor separado será adicionado ao final de cada linha selecionada.

Paleta de Comandos

Paleta de Comandos

  • No Windows: Ctrl + p
  • No Mac: Command + p
  • No Ubuntu: Ctrl + p

Acesse todos os comandos disponíveis com base no seu contexto atual.

Alguns dos mais utilizados são:

  1. Abrir um arquivo

Para ir para um arquivo, use o comando acima e digite o nome do arquivo que você está procurando. Isso deve ajudá-lo a localizar arquivos rapidamente.

  1. Consulte o comando de referência do teclado

Todos os comandos estão na Paleta de Comandos com a ligação de chave associada (se existir). Se você esquecer um atalho de teclado, use a Paleta de Comandos para ajudá-lo.
